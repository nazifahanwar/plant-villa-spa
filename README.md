## Features

- Browse and explore a variety of indoor plants
- Plant care guides with tips on watering, sunlight, and fertilization
- Book consultations with plant care experts
- Newsletter subscription for updates and tips
- User authentication with email/password and Google Sign-In
- Profile update with display name and photo
- Responsive design for all screen sizes
- Smooth UI interactions with animations

---

## Tech Stack

- **Frontend:** React.js  
- **Routing:** React Router  
- **UI & Styling:** Tailwind CSS, DaisyUI  
- **Icons:** React Icons  
- **Notifications:** Sonner (toast notifications)  
- **Animations & Sliders:** Framer Motion, Swiper.js  
- **Hosting & Backend:** Firebase (Authentication, Hosting)
